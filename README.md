# Vanilla App Template

An interactive image gallery app built with vanilla JavaScript. Features dynamic searching, infinite scroll loading for a seamless browsing experience.

## :sparkles: Features

  * Clean, minimal design focused on showcasing visuals
  * Asynchronous data fetching + pagination from pixabay
  * Lightbox effect for viewing images in full size
